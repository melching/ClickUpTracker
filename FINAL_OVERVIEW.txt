HOW TO USE IT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

QUICK START (5 minutes):

1. Install dependencies:
   $ pip install -r requirements.txt

2. Get your ClickUp API token:
   • Go to ClickUp → Settings → Apps
   • Click "Generate" under API Token
   • Copy the token

3. Run the app:
   $ python3 clickup_tracker.py

4. Configure (first time only):
   • Click the ⏱️ icon in menubar
   • Click "⚙️ Settings"
   • Paste your API token
   • Click "Save"

5. Start tracking:
   • Click "🎯 Assign Task" (optional)
   • Click "▶️ Start Tracking"
   • Work on your task
   • Click "⏸ Pause/Stop Tracking" when done

That's it! Your time is now tracked in ClickUp.
